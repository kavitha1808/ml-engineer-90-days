# Linear Regression using Gradient Descent

## Project Overview

This project implements Linear Regression from scratch using Gradient Descent.

The goal is to predict customer spending based on marketing budget.

This example demonstrates how businesses can estimate revenue generated from marketing investments.

## Problem Statement

Marketing teams invest money in campaigns such as:

- Digital ads
- Social media promotion
- Email campaigns

This project predicts how much customers are likely to spend based on the marketing budget.

## Dataset

Features:

marketing_budget – money spent on marketing

Target:

customer_spending – predicted customer spending

## Algorithm

Linear Regression with Gradient Descent.

Model equation:

y = wx + b

Where

w = weight  
b = bias

Cost function used:

Mean Squared Error (MSE)

Gradient Descent updates parameters to minimize error.

## Technologies Used

Python  
NumPy  
Pandas  
Matplotlib

## Output

The model generates a regression line showing the relationship between marketing budget and customer spending.
