## 🔧 Feature Extraction Techniques Used

1️⃣ One-Hot Encoding  
Converts categorical variable (city) into numerical format.

2️⃣ TF-IDF Vectorization  
Converts raw text into numerical vectors.

3️⃣ PCA (Principal Component Analysis)  
Reduces feature dimensions while preserving maximum variance.

Run:
python feature_extraction.py
