# Student Enrollment & Attendance System

This mini project demonstrates practical usage of:
- List (to store enrolled students)
- Tuple (to store immutable student records)
- Set (to manage attendance without duplicates)

## Features
- Enroll students
- Mark attendance
- Display enrolled students
- Display present students

## Concepts Used
- Data structures
- Membership checking
- Immutability vs mutability

## Learning Outcome
Understanding **when and why** to use:
- List
- Tuple
- Set
