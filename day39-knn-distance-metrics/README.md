# Day 39 - K-Nearest Neighbors (KNN) and Distance Metrics

## 📌 Objective
This project demonstrates the basics of the **K-Nearest Neighbors (KNN)** algorithm and the concept of **distance metrics** used in KNN.

---

## 📚 Topics Covered
- KNN classification
- Euclidean distance
- Manhattan distance
- Minkowski distance
- Feature scaling importance
- KNN decision boundary visualization

## 🧠 Key Learnings

KNN is a simple and powerful supervised learning algorithm.

It predicts based on the majority vote of nearest neighbors.

Distance metrics strongly affect performance.

Feature scaling is very important for KNN.

Euclidean distance is most common, but Manhattan and Minkowski are also useful.

## ✅ Conclusion

KNN is one of the easiest machine learning algorithms to understand. It works well for small datasets and is useful for learning classification fundamentals. Choosing the right K and proper distance metric can greatly improve performance.
