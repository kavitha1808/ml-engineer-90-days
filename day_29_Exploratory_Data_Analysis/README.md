# 📊 Student Performance Analysis – EDA Project (.py Version)

## 📌 Overview
This project performs Exploratory Data Analysis (EDA) using pure Python scripts (no Jupyter Notebook).

The objective is to analyze factors affecting student marks.

---

## 📂 Project Structure

Day_29_EDA_Project/
│
├── data/
├── outputs/
├── src/
├── main.py
└── README.md

---

## 🔍 Steps Performed

1. Data loading
2. Basic dataset inspection
3. Missing value check
4. Statistical summary
5. Histogram visualization
6. Boxplot (outlier detection)
7. Scatter plot analysis
8. Correlation heatmap

---

## 📊 Key Insights

- Strong positive correlation between Hours_Studied and Marks
- Moderate relationship between Sleep_Hours and Marks
- No missing values found
- No extreme outliers detected

---

## 🛠 Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn

---

## ▶ How to Run

1. Install dependencies:

pip install pandas matplotlib seaborn

2. Run the script:

python main.py

outputs/plots/
