# Day 45 - Model Selection Strategies

## Overview

This project demonstrates essential model selection techniques using scikit-learn on the Iris dataset.

## Features

- Train-Test Split
- K-Fold Cross Validation
- Hyperparameter Tuning using GridSearchCV
- Model Evaluation

## Tech Stack

- Python
- scikit-learn

## Installation

pip install -r requirements.txt

## Usage

python model_selection.py

## Output

- Cross-validation scores
- Best hyperparameters
- Test accuracy

## Learning Outcome

Understand how to select and tune machine learning models effectively.
