# Day 17 – NumPy Arrays

## Overview

- This project covers the fundamentals of NumPy arrays, including:

- Array creation

- Array operations

- Broadcasting

- Practice problems

- Simple mini projects


All programs use vectorized NumPy operations (no loops).
